# 📚 Data-Structure-Learning (Java)

本项目是我个人学习**数据结构**的实践代码库，使用 **Java** 语言从零实现常见数据结构，并配有详细的学习文档。  
目标是深入理解每种结构的内部原理，同时也为其他学习者提供可运行的示例和笔记。

## 📁 项目结构
.
├── src/
│ ├── linear_list/ # 线性表
│ │ ├── SeqList.java # 顺序表
│ │ ├── LinkedList.java # 单链表
│ │ └── DoublyLinkedList.java # 双链表（可选）
│ ├── queue/ # 队列
│ │ ├── ArrayQueue.java # 顺序队列
│ │ ├── LinkedQueue.java # 链队列
│ │ └── CircularQueue.java # 循环队列
│ ├── string/ # 串
│ │ ├── KMP.java # KMP 模式匹配
│ │ └── MyString.java # 串的基本操作
│ ├── tree/ # 树
│ │ ├── BinaryTree.java # 二叉树（递归/非递归遍历）
│ │ ├── BST.java # 二叉搜索树
│ │ └── Heap.java # 堆（大/小顶堆）
│ └── Main.java # 可选：综合演示入口
├── docs/ # 学习文档（Markdown）
│ ├── linear_list.md
│ ├── queue.md
│ ├── string.md
│ └── tree.md
└── README.md
